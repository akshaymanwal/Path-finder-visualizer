# Pathfinding Visualizer 🚀

An interactive web-based pathfinding visualizer to understand algorithms visually.

## 🔥 Features
- BFS, DFS, Dijkstra, A*
- Animated grid visualization
- Algorithm summary with tabs:
  - Info
  - Complexity
  - Example
  - Weakness
- Clean UI with modern design

## 🧠 Algorithms Covered
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra’s Algorithm
- A* Algorithm

## 💻 Tech Used
- HTML
- CSS
- JavaScript
